This is an *HR Analytics Dashboard* built using *Microsoft Excel* to visualize key Human Resources metrics and performance indicators.

##  Overview
The dashboard provides insights into:
- Total number of employees  
- Average working hours  
- Total salaries and payroll costs  
- Overtime hours by department  
- Comparison between overtime hours and employee count  
- Departmental distribution of workforce and workload  

##  Purpose
The goal of this dashboard is to *analyze employee productivity, department efficiency, and workload distribution* in a simple, visual, and data-driven way.  

It helps HR teams and management monitor trends, identify imbalances between departments, and make better decisions regarding staffing and overtime management.
